# Advanced Engineering Mathematics (AEM)

## Description

This is my repository for the Advanced Engineering Mathematics class which will contain files and activities for the subject. It includes comprehensive lecture materials, examples, and exercises covering various topics in engineering mathematics.

## Contents

- **MODULE 1**: Series (Taylor series, power series, and more)
- **MODULE 2**: Series Solutions (Solutions About Ordinary Points , Power Series Solutions, and more)
- **MODULE 3**: Vectors (Vectors, Dot Product, Cross Product and more)
- **MODULE 4**: Matrices (Matrix Addition, Matrix Multiplication, Solving Systems of equations and more)
- **MODULE 5**: Vector Calculus (Vector Functions, Partial Dervatives, Curl and Divergence, and more)
- **Reference Books**: Additional resources and references

## About

This repository serves as a comprehensive learning resource for students taking the Advanced Engineering Mathematics course. Each module contains detailed explanations, worked examples, and practice problems.
